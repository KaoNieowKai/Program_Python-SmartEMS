# SmartEMS - ระบบจัดการสุขภาพแบบอัจฉริยะ

ระบบบริหารจัดการข้อมูลสุขภาพและเอกลักษณ์บุคคลแบบครบวงจร ที่รวมเทคโนโลยีการอ่านบัตรประชาชนไทย และการติดตามสัญญาณชีพวิตาลในเวลาจริง

---

## 📋 ส่วนประกอบของโครงการ

### 1. **SmartEMS** (แอปพลิเคชันเดสก์ทอป)
- เขียนด้วย **Tkinter** และ **OpenCV**
- ฟีเจอร์หลัก:
  - 📱 อ่านข้อมูลจากบัตรประชาชนไทย
  - 💓 ตรวจสอบชีพจร และค่าออกซิเจน (MAX30102)
  - 📷 บันทึกรูปภาพผ่านกล้อง
  - 📄 สร้างรายงาน PDF
  - 🔔 ตรวจสอบการแจ้งเหตุจากเว็บแอปพลิเคชัน

### 2. **python_ThaiCID-GUI-PyQT6-main** (GUI อ่านบัตร)
- เขียนด้วย **PyQt6**
- โปรแกรมเสริมสำหรับอ่านข้อมูลบัตรประชาชนไทยอย่างเฉพาะเจาะจง
- UI ออกแบบด้วย Qt Designer

### 3. **SmartEMS-WebApp** (เว็บแอปพลิเคชัน)
- เขียนด้วย **FastAPI**
- ฟีเจอร์:
  - 📊 แสดงข้อมูลผู้ป่วยและสถิติ
  - 📤 อัปโหลดและบริหารรายงาน
  - 📲 ส่งการแจ้งเหตุไปยังแอปพลิเคชัน
  - 💾 บันทึกข้อมูลใน SQLite

## 🚀 การติดตั้งและใช้งาน

### ความต้องการ
- Python 3.8+
- Tkinter (มาพร้อมกับ Python)
- PyQt6 (สำหรับ GUI หากใช้งาน)
- ฮาร์ดแวร์: MAX30102 เซนเซอร์, กล้องเว็บแคม หรือ Picamera2

### การติดตั้ง
```bash
# สร้าง virtual environment
python -m venv venv
venv\Scripts\activate

# ติดตั้ง dependencies
pip install -r requirements.txt
```

### การรันแอปพลิเคชัน

**แอปพลิเคชันเดสก์ทอป:**
```bash
python SmartEMS.py
```

**เว็บแอปพลิเคชัน:**
```bash
cd SmartEMS-WebApp
uvicorn main:app --reload
# เข้าแล้วไปที่ http://localhost:8000
```

## 📦 ไลบรารีหลัก

- **tkinter** - GUI แอปพลิเคชัน
- **PyQt6** - GUI สำหรับ Thai ID Reader
- **OpenCV** - ประมวลผลภาพและบันทึกวิดีโอ
- **reportlab** - สร้างเอกสาร PDF
- **FastAPI** - เว็บเฟรมเวิร์ก
- **Pillow** - ประมวลผลภาพ
- **pywin32** - การรองรับ Windows

---

## 🔌 การเชื่อมต่อระหว่างส่วนประกอบ

- **SmartEMS** ↔ **SmartEMS-WebApp**: อ่านและส่งไฟล์การแจ้งเหตุ (`alerts/alert_latest.txt`)
- **ThaiCIDReader.py**: โมดูลที่ใช้ร่วมกันสำหรับอ่านบัตรประชาชน
- **ฐานข้อมูล**: SQLite ที่เก็บข้อมูลผู้ป่วยและรายงาน
